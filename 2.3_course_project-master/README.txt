README
    CIS453L Course Project - Toronto Ccupcake Redesign
    This webpage is a wbpage, created with react and uses single page
    navigation that is used for navigating between the diffeent 
    sections and catagories of the toronto cupcake website. It also uses 
    custome components for images as well as other HTML elements.
    This prototype was updated accordingly with feedback from
    peer review.

Author
    Christopher Rocca

Install
    To install this prototype, naviage to 
        https://github.com/ChrisRocca3/2.3_course_project
    Run 'npm install' in command while in the projects directory
    to install the node modules.
    Run 'npm start' in command while in the projects directory.
